# Learning PlayWright

This repository contains Playwright learning examples and test code for demo/learning purposes.

## Contents
- `01_Chapter 1/` — basic JS samples used for initial setup and verification.
- `02_Chapter2_Java_Concepts/` — placeholder for Java concept files.

## Requirements
- Node.js (>=16)

## Setup
1. Install dependencies:

```bash
npm install
```

2. Install Playwright browsers:

```bash
npx playwright install
```

## Running tests

Run Playwright tests:

```bash
npx playwright test
```

View HTML report after tests:

```bash
npx playwright show-report
```
